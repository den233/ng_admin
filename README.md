# typerx (Typescript Rest extra)
A lightweight typescript annotation rest based extra (express、 mongoose、 angular、ng-zorro、ng-alain ...).

Automatic maintenance rest api and flexible.

[中文 README](README-zh_CN.md)


# module features

### system management
- settings 
- accounts  
- menus 
- roles
- dictionary
- logs

### CMS management
- pages
- articles
- categories
- media
- widgets

### Quick start

```bash
# clone our repo
# --depth 1 removes all but one .git commit history
$ git clone --depth 1 https://github.com/vellengs/typerx.git

# change directory to our repo
$ cd typerx

# install the repo with npm
$ npm install

# start the server
$ npm start

```

### Test

```
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
